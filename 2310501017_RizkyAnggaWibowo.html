Pengalaman Saya

Awal mula saya belajar tentang pemrograman pada saat saya SMK, di SMK bahasa pemrograman yang saya pelajari adalah HTML, dengan bahasa pemrograman HTML saya belajar membuat table, list, memasukan gambar dll. Setelah itu saat saya kelas 2 SMK baru mulai belajar bahasa pemrograman baru yaitu menggunakan CSS, saya belajar memberikan warna, mengatur posisi, mengatur ukuran, dan lain-lain agar website yang saya buat lebih menarik menggunakan CSS

JavaScript adalah bahasa pemrograman yang digunakan agar membuat halaman web yang sudah saya buat lebih interaktif. Dengan JavaScript, saya bisa mengubah tampilan halaman, merespons aksi pengguna seperti klik, mengelola data, dan berinteraksi dengan server tanpa memuat ulang halaman. Ini membantu dalam membuat aplikasi web yang lebih menarik dan responsif.
